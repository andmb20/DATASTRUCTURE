RESUMO

Arrays, Pilhas, Filas podem ser implementados com arrays nativos ou classes.

Listas Ligadas, Árvores, Grafos geralmente usam classes com nós.

Conjuntos (Set), Dicionários (Map) já têm implementações prontas em JavaScript.

Arrays (Vetores)

Em JavaScript, arrays são listas ordenadas que podem guardar vários valores (de qualquer tipo). Eles são dinâmicos (crescem e diminuem automaticamente).

let frutas = ["maçã", "banana", "laranja"]; // Criando um array
frutas.push("uva"); // Adiciona no final
frutas.pop(); // Remove do final

Pilhas (Stacks)

Pilhas seguem o princípio LIFO (Last In, First Out). Podemos criar uma pilha usando um array ou uma classe.

Usando Array:

let pilha = [];
pilha.push(1); // Empilha
pilha.push(2);
let topo = pilha.pop(); // Desempilha (2)

Usando Classe:

class Pilha {
  constructor() {
    this.itens = [];
  }
  empilhar(item) {
    this.itens.push(item);
  }
  desempilhar() {
    return this.itens.pop();
  }
}

Filas (Queues)

Filas seguem o princípio FIFO (First In, First Out) Podemos criar uma fila usando um array ou uma classe.

Usando Array:

let fila = [];
fila.push(1); // Enfileira
fila.push(2);
let primeiro = fila.shift(); // Desenfileira (1)

Usando Classe:

class Fila {
  constructor() {
    this.itens = [];
  }
  enfileirar(item) {
    this.itens.push(item);
  }
  desenfileirar() {
    return this.itens.shift();
  }
}

Listas Ligadas (Linked Lists)

Cada elemento (nó) tem um valor e um ponteiro para o próximo nó

class No {
  constructor(valor) {
    this.valor = valor;
    this.proximo = null;
  }
}

class ListaLigada {
  constructor() {
    this.cabeca = null;
  }
  adicionar(valor) {
    let no = new No(valor);
    if (!this.cabeca) {
      this.cabeca = no;
    } else {
      let atual = this.cabeca;
      while (atual.proximo) {
      atual = atual.proximo;
    }
    atual.proximo = no;
    }
  }
}

Conjuntos (Sets)

São coleções de valores únicos (sem repetições).

let conjunto = new Set();
conjunto.add(1);
conjunto.add(2);
conjunto.add(1); // Não adiciona repetido
console.log(conjunto.has(1)); // true

Dicionários (Objetos e Maps)

Armazenam pares chave-valor.

Usando Objeto:

let dicionario = { nome: "João", idade: 12 };
console.log(dicionario.nome); // "João"

Usando Map (melhor para chaves dinâmicas):

let mapa = new Map();
mapa.set("nome", "Maria");
mapa.set("idade", 12);
console.log(mapa.get("nome")); // "Maria"

Árvores (Trees)

Estrutura hierárquica com nós (pai e filhos).

class NoArvore {
  constructor(valor) {
    this.valor = valor;
    this.esquerda = null;
    this.direita = null;
  }
}

let raiz = new NoArvore(10);
raiz.esquerda = new NoArvore(5);
raiz.direita = new NoArvore(15);

Grafos (Graphs)

Conjunto de nós conectados por arestas.

class Grafo {
  constructor() {
    this.vertices = new Map(); // Chave: nó, Valor: lista de adjacências
  }
  adicionarVertice(vertice) {
    this.vertices.set(vertice, []);
  }
  adicionarAresta(origem, destino) {
    this.vertices.get(origem).push(destino);
    this.vertices.get(destino).push(origem); // Para grafos não direcionados
  }
}
