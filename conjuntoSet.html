Conjuntos ( Sets )

"Conjunto é como um clube exclusivo: só entra quem é único e não se repete!"

Um conjunto (Set) é uma estrutura de dados que armazena elementos únicos e não ordenados, sem índices. Ideal para garantir que não haja duplicatas.

    Características Principais

  1. Unicidade: Não permite elementos repetidos.
  2. Não ordenado: A ordem de inserção não é preservada.
  3. Operações matemáticas: Suporta união, interseção e diferença.
  4. Otimizado para buscas: Verifica a existência de elementos rapidamente. Verifica a existência de elementos em tempo constante (em implementações baseadas em hash).

Exemplos em Diferentes Linguagens

// JavaScript
const set = new Set();
set.add(1);
set.add(2);
set.add(1); // Ignorado (duplicado)
console.log(set.has(1)); // true
set.delete(2);


# Python
meu_set = {1, 2, 3} # ou set([1, 2, 3])
meu_set.add(4)
meu_set.discard(2)
print(3 in meu_set) # True


Complexidade das Operações

Operação Complexidade Exemplo em JavaScript
add() O(1) set.add(5)
delete() O(1) set.delete(5)
has() O(1) set.has(5)
union() O(n + m) set1.union(set2)

Assume implementação com hash tables, como em Python e JavaScript.

    Quando Usar Conjuntos ( Set )?

    Casos Ideais:

  • Remover duplicatas de uma lista.
  • Verificar existência rápida de elementos (ex.: bloqueio de usuários).
  • Operações matemáticas (união/interseção).
  • Rastrear itens únicos (ex.: palavras únicas em um texto).

    Dicas de Uso Avançado

  1. Converta listas para sets para remover duplicatas:

      2. lista = [1, 2, 2, 3]
      sem_duplicatas = list(set(lista)) # [1, 2, 3]

  2. Use operações de conjuntos para comparar dados:

      3. const admins = new Set(["Alice", "Bob"]);
      const users = new Set(["Alice", "Charlie"]);
      const adminsComuns = new Set([...admins].filter(x => users.has(x))); // {"Alice"}

  3. Sets com objetos: Em JavaScript, use propriedades únicas (ex.: IDs), pois objetos são comparados por referência.

    Analogia Prática

    Pense em um cartão de visitas digital:

  • Você pode adicionar muitos contatos, mas cada e-mail só aparece uma vez.
  • Não importa a ordem em que foram adicionados.

    Quando NÃO Usar:

  • Se precisar de ordem (use arrays/listas).
  • Se precisar de acesso por índice (use arrays/listas).

Dica Importante

Em JavaScript, Set não tem método para acesso direto por índice. Use Array.from(set)[0] se necessário.
