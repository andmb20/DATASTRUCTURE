Pilha (Stack)

"Pilha é como uma pilha de pratos: o último que entra é o primeiro que sai!"

Uma pilha (stack) é uma estrutura de dados linear que segue o princípio LIFO (Last In, First Out), onde o último elemento adicionado é o primeiro a ser removido.

    Características Principais

  1. LIFO: Último elemento inserido é o primeiro a ser removido.
  2. Operações Básicas:
    • push(): Adiciona um elemento no topo.
    • pop(): Remove o elemento do topo.
    • peek()(ou top()): Retorna o elemento do topo sem remover.
  3. Tamanho Dinâmico: Cresce ou reduz conforme elementos são adicionados/removidos.
  4. Acesso Restrito: Só o elemento do topo é acessível diretamente.

Exemplos em Diferentes Linguagens

// JavaScript (usando array como pilha)
let stack = [];
stack.push(1); // [1]
stack.push(2); // [1, 2]
stack.pop(); // Remove 2 → [1]


# Python (usando list)
stack = []
stack.append(3) # [3]
stack.append(4) # [3, 4]
stack.pop() # Remove 4 → [3]

// Java (Stack classe)
Stack<Integer> stack = new Stack<>();
stack.push(5); // [5]
stack.peek(); // 5


Complexidade das Operações

Operação Complexidade Exemplo em JavaScript
push() O(1) stack.push(10)
pop() O(1) stack.pop()
peek() O(1) stack.peek()
isEmpty() O(1) stack.isEmpty()

    Quando Usar Pilhas?

    Casos Ideais:

  • Undo/Redo em editores de texto (última ação é a primeira a ser desfeita).
  • Navegação de páginas (voltar/avançar no histórico do navegador).
  • Validação de parênteses em expressões matemáticas.
  • Algoritmos recursivos (simulação de call stack).

    Dicas de Uso Avançado

  1. Pilhas com Tamanho Fixo:
    • Evite estouro de memória definindo um limite máximo.
  2. Pilhas Múltiplas:
    • Útil para gerenciar estados em aplicações complexas (ex.: jogos).
  3. Pilhas em Algoritmos:
    • DFS (Busca em Profundidade) usa pilha para rastrear nós.

    Analogia Prática

    Pense em uma pilha de livros:

  • Você só pode adicionar ou remover o livro do topo.
  • Para pegar o livro do fundo, precisa remover todos acima!

    Quando NÃO Usar:

  • Se precisar de acesso aleatório a elementos (use arrays).
  • Se a ordem de remoção deve ser FIFO (use filas/queues).

Dica Importante

SEMPRE verifique se a pilha está vazia antes de chamar pop() ou peek() para evitar erros!
