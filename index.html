Data Structure - REVIEW

    Tipos Primitivos (básicos):

  1. Inteiros (int) - números sem decimal:

    2. 3, -5, 42

  2. Float (números decimais):

    3. 3.14, -0.001

  3. Texto (string/caracteres):

    4. "Olá", "A" (uma letra também é texto)

  4. Boolean (verdadeiro/falso):

    5. True, False

ARRAY PILHA (STACK) FILA (QUEUE)

    Estruturas de Dados Básicas (compostas):

  1. Array/Vetor/Lista: Coleção ordenada de itens

    2. Ex.: [1, 2, 3] ou ["maçã", "banana"]

  2. Struct/Registro: Agrupa diferentes tipos em um "pacote"

    3. Ex.: Um registro Aluno com nome (texto) e nota (float)

    [ {"nome": "João", "nota": 7.5}, {"nome": "Maria", "nota": 9.3} ]

  3. String: Na verdade, é um array de caracteres!

    4. ("ABC" = ['A','B','C'])
